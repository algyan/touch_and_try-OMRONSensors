## <a name="S-0">説明</a>

本テキストは、種々の環境データを簡単に測定できる「OMRON環境センサ」をAzureに接続し、IoTの基礎から応用までを学ぶタッチ＆トライです。

初心者にも優しいnode.jsを使い、Bluetooth Low Energy (BLE)の基本的な接続方法からAzure接続の基礎までを学べる初級編と、IoT Edgeの機能を使った本格的なIoTシステムの構築を体験する上級編に分かれています。
